ドローン産業トレンド分析

無人航空機の飛行計画（申請）から、飛行目的・機体構成・資格保有の分布と月次推移を定量把握するダッシュボードです。

申請ベース（実飛行・実態ではありません） データ品質は公式に非保証 対象期間: 読込中…

このダッシュボードの読み方

  • 「飛行計画（申請）」の集計です。実際の飛行や運航実態を表すものではありません。
  • 元データは紙資料のスキャンから抽出されており、完全性・正確性は公式に保証されていません。傾向把握の参考としてご利用ください。
  • 日時データに品質上の問題があるため、月次の集計単位は「ファイル＝対象月」を信頼単位としています。
  • 飛行目的を多数同時に申請した計画（包括申請）は、目的別構成の分布から除外し、別途その件数・割合を示しています。
  • 1つの飛行計画が複数の機体・飛行エリアにわたって複数行に展開されるため、飛行目的・資格・空域・方法は飛行計画単位、機体構成は機体（airframe）単位で重複排除して集計しています。
  • 出発地は市区町村重心レベルに秘匿化済みです。本サイトでは粒度を上げる二次加工を行っていません。

月次の飛行計画件数推移

各月の飛行計画数（重複排除後）。年度をまたぐ12ヶ月の申請ボリュームの推移を示します。

飛行目的の構成（業務）

包括申請を除く。1件で複数目的を持つため合計は件数と一致しません。

機体の種類

申請に用いられた機体種別の構成（機体単位・重複排除後）。

資格・認証の保有率

各資格を保有する飛行計画の割合（全件に対する比率）。

製造区分・改造の有無

機体の製造区分（左）と改造の有無（右）。

主要な飛行目的の月次推移

上位の飛行目的について、各月の構成比（包括申請を除く目的フラグ合計に対する割合）の推移を示します。

資格保有率の月次推移

各月の全件に対する資格保有割合の推移。資格制度の浸透傾向の把握に。

飛行空域の構成

DID・150m以上・空港周辺など該当空域（複数該当あり）。

飛行方法の構成

夜間・目視外・催し物上空など飛行方法（複数該当あり）。

出発地（都道府県）上位

出発地テキストから抽出した都道府県別の件数上位15。市区町村重心レベルに秘匿化済みのデータに基づきます。
